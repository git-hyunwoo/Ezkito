FAQ

Is EzKito free?

Yes, EzKito is completely free to use and does not require any signup.

Are my files safe?

Files are processed on the server only for the duration of the conversion. EzKito does not keep or share your files.

Can I convert multiple files at once?

Yes, you can upload multiple files at once. For some conversions, EzKito will return a ZIP file containing all results.
